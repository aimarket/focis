Airsoft Turret

This project combines many skills such as 3D CAD, Python, Linux, and computer engineering.

A raspberry-pi model B is used for both function and video recgonition using OPEN-CV for face detection. Other components used include Stepper motor, Servo motor, LED Lights, voltage regulator, MOSFET, and an HD Camera module.
